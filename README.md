# Save-My-Lab
A website that simulates a seat reservation system in DLSU computer laboratories.

## 1. Clone the Repository

```bash
git clone https://github.com/YUANQTIE/Save-My-Lab.git
cd Save-My-Lab
```

## 2. Install Dependencies

```bash
npm install express express-fileupload hbs mongodb mongoose
```

## 3. Run for the tables to show up in compass

```bash
node index.js
```

## 4. Upload the json files in 'uploadmeincompass'

```bash
cd database/uploadmeincompass
```

## 5. Refresh and Check the Website

```bash
Go to a browser and input localhost:3000. Enter.
```
